# BGTAT-Final-Project
BGTAT: Physics-Aware AI Dashboard for Gas Turbine Diagnostics Description: Developed a bespoke predictive maintenance platform (BGTAT) specifically for GE Frames 5, 6, 7, and 9. Unlike traditional "black-box" models, this tool integrates hardware-specific physics—specifically load-dependent Exhaust Swirl Mapping—to detect combustion anomalies.
